# Asteroids

It's just a game for training AI models. We'll export data in CSV files.

## Neural Network

We'll use a neural network to train the AI models. The neural network will have 8 inputs and 5 outputs.

Inputs:
- Distance to the nearest asteroid in front
- Angle to the nearest asteroid in front
- Distance to the nearest asteroid on the right
- Angle to the nearest asteroid on the right
- Distance to the nearest asteroid on the left
- Angle to the nearest asteroid on the left
- Distance to the nearest asteroid behind
- Angle to the nearest asteroid behind

Outputs:
- Move forward
- Move backward
- Rotate left
- Rotate right
- Shoot
